Objects

How handy are they!

Key Notes

  • Variables are cool, but they only hold one value at time, so, sometimes you need to hold more complex data than just numbers or single strings. That's where objects are in action. They can hold a group of data.
  • Its important to take in mind that there are two types of objects: Native and Host
  • Native objects are part of JavaScript, and in a more board way to say it, part of the ECMAScript standard. Or if you want to get more technical, are "built-in" in JS language
  • Example of Native objects are Array, Promise, JSON
  • Host objects are part of the environment that your code runs in, like in a server, desktop client or mobile client.
  • Example of Host objects are Window, document, history...
  • Think an object as a 'thing', like in real life, objects in real life have detail... like color, size. In JS the 'details' of an object are called properties
  • To access to a property of an object use the dot notation, like nameOfObject.property
  • So, if you want to save a value of a property of an object you will do, for example, the title of the current page let value = document.title; (See Demo 1)
  • To update a value of an object, you can do it like accessing to the property and the assignt it a new value like you would to a normal variable. document.title = 'lol surprise'; (See demo 1)
  • Objects has methods too. This methods are functions that work, process information using the properties like for example, if you want to find something in your html document searching by ID you would do let myIdElement = document.getElementById('element');
  • As you can see, methods has parameters, that's the stuff that goes between '()', and is a way to give methods something to work with, if needed!
  • Arrays are an special native object of JS. They hold a group of data as stack plates, each plate represent an element or data. From here, all code discussed here you can see them in action on Demo 2!
  • There are two ways of declare them (arrays)! let myArray1 = new Array(); or let myArray2 = [];
  • There are two ways of adding data to them! myArray1.push(1); or using the index myArray2[0] = 1;
  • You can get the length of an array using the method myArray1.length;
  • To going through all the content of an array, you can use the function Array.forEach() Example: 
     
                    myArray.forEach((value, index, array) => {
                        console.log(value); //current value
                        console.log(index); //current index
                        console.log(array); //entire array
                    });
  • To filter the array and create a new Array with the filtered content you can use Array.filter() Example (filtering odd numbers): 
                            let myArray2oddNumbers = myArray2.filter((value,index,array)=>{
                            if(value % 2){
                                console.log('myArray2[' + index + '] = ' + value);
                                return value;
                            }
                        });
  • To modify each element of the array you can use Array.map() Example (adding +5 to each element): 
                            let myArray1plus5 = myArray1.map((value,currentValue,currentIndex,array) => {
                            return value + 5;
                        });
  • To create a single value by adding all values of the array you can use Array.reduce() Example: 
                            const initialValue = 0;
                        let myArray2Single = myArray2.reduce((accomulator,currentValue)=> accomulator + currentValue,initialValue);
